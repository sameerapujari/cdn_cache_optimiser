# CDN Cache Optimizer

CDN Cache Optimizer is a simulation-based tool that compares different cache eviction strategies used in Content Delivery Networks (CDNs). The system evaluates the performance of **Least Recently Used (LRU)** and **Least Frequently Used (LFU)** caching algorithms under simulated web traffic.

The application generates request patterns and measures key performance metrics such as **cache hit ratio** and **average latency**, helping analyze which caching strategy performs better under different conditions.

## Features
- Simulation of CDN caching behavior
- Comparison of **LRU vs LFU** eviction algorithms
- Adjustable simulation parameters
- Performance metrics visualization using Streamlit

## Tech Stack
- Python
- Streamlit
- Pandas
